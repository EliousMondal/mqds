Evaluates an intermediate sum over system basis states using a Monte-Carlo importance sampling procedure by building a probability density function based on the magnitude of all of the density matrix elements and uniformly sampling the cumulative distribution function to select out the "most important" reduced density matrix element as the initially-occupied state for subsequent propagation.

Where \( m_{n_t n'_t} = \frac{r_{n_t n'_t}}{\sum_{n_tn'_t} r_{n_t n'_t} }\) and \( \xi \) is a uniform [0,1) random number. Performing this accumulates a Monte-Carlo weight of \( e^{i(\theta_{n_t^o} - \theta_{n_t^{'o}})} \sum_{n_tn'_t} r_{n_t n'_t} \)
